## 🚀 Featured Project — India Inflation ETL Pipeline

> Automated ETL pipeline tracking India's CPI inflation (2000–present) with GitHub Actions monthly scheduler

- Auto-downloads FRED API data → Cleans & transforms → Loads to SQLite → Power BI dashboard
- Runs automatically every 1st of the month via GitHub Actions — zero manual effort
- Found: 123/303 months (40%) exceeded RBI's 6% inflation limit · 2010s worst decade at 7.34%
